Project information

  • Project Title: Decentralized file storage for optimized cloud storage
  • Tech Stack:
    • Firebase for User Authentication
    • IPFS for the decentralized network architecture
    • Progressive Web App (PWA) (HTML, CSS & JS) for building an interface that allows users to upload and view their files and also to show a global view of all the peers using the network.
  • Description:
    • The decentralized storage network was built using the IPFS infrastructure.
    • Using this architecture, chunks of the media file were stored in multiple nodes on the network.
    • When the user requests for a certain media file, be it text, an image or a video, all these chunks of data were retrieved from these nodes in parallel thereby reducing the retrieval time drastically.
    • A Progressive Web App (PWA) was built that enables a cross platform interface for uploading and listing the user's files. In simpler terms, it was a universal decentralized cloud files storage platform.
    • Downloads speeds were reduced by almost 80%.
